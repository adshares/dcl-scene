Adshares example scene for **Decentraland**. You can use it to learn how to integrate with Adshares protocol.

See [DCL Plugin](https://github.com/adshares/decentraland).

## Install

```bash
npm i -g decentraland
npm install
```

## Try it out

```bash
dcl start
```
